# 03. 光照模型基础：让物体有立体感

如果物体只有一种颜色，它看起来就像一张纸片。光照是让物体产生立体感的关键。
本章我们将学习最经典的 **Blinn-Phong 光照模型**。

## 核心概念

1.  **法线 (Normal)**：垂直于物体表面的方向向量。决定了面朝向哪里。
2.  **光照方向 (Light Direction)**：光线射入的方向（通常取反方向，即指向光源）。
3.  **点积 (Dot Product)**：衡量两个向量的相似程度。
    *   `dot(N, L)`：如果结果为 1，说明光线垂直照射（最亮）；如果结果 <= 0，说明背光（全黑）。

---

## 1. 漫反射 (Diffuse)

漫反射模拟的是粗糙表面的光照，光线均匀向四周散射。
公式：`Diffuse = 材质颜色 * 光源颜色 * max(0, dot(法线, 光方向))`

### 步骤
1.  **获取法线**：在顶点着色器中，把模型法线转换到世界空间。
2.  **获取光方向**：Unity 内置变量 `_WorldSpaceLightPos0`。
3.  **计算点积**：在片元着色器中计算 `saturate(dot(N, L))`。

## 2. 高光反射 (Specular)

高光模拟光滑表面的反光点。
Blinn-Phong 模型引入了 **半程向量 (Half Vector)** `H`，它是光方向 `L` 和视线方向 `V` 的中间向量。
公式：`Specular = 光源颜色 * pow(max(0, dot(法线, 半程向量)), 光泽度)`

---

## 实战：编写 Blinn-Phong Shader

创建一个新 Shader `03_BlinnPhong`，输入以下代码：

```shader
Shader "MyShader/03_BlinnPhong" {
    Properties {
        _Diffuse ("Diffuse Color", Color) = (1, 1, 1, 1) // 漫反射颜色
        _Specular ("Specular Color", Color) = (1, 1, 1, 1) // 高光颜色
        _Gloss ("Gloss", Range(8, 256)) = 20 // 光泽度（越大光斑越小）
    }
    
    SubShader {
        Tags { "LightMode"="ForwardBase" } // 重要：指定光照模式
        
        Pass {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            
            #include "UnityCG.cginc"
            #include "Lighting.cginc" // 引入内置光照变量
            
            struct appdata {
                float4 vertex : POSITION;
                float3 normal : NORMAL; // 需要法线信息
            };
            
            struct v2f {
                float4 pos : SV_POSITION;
                float3 worldNormal : TEXCOORD0; // 传递世界空间法线
                float3 worldPos : TEXCOORD1;    // 传递世界空间位置（用于计算视线）
            };
            
            fixed4 _Diffuse;
            fixed4 _Specular;
            float _Gloss;
            
            v2f vert (appdata v) {
                v2f o;
                o.pos = UnityObjectToClipPos(v.vertex);
                
                // 1. 法线变换：模型空间 -> 世界空间
                // UnityObjectToWorldNormal 是内置函数，处理了缩放等复杂情况
                o.worldNormal = UnityObjectToWorldNormal(v.normal);
                
                // 2. 顶点位置变换：模型空间 -> 世界空间
                o.worldPos = mul(unity_ObjectToWorld, v.vertex).xyz;
                
                return o;
            }
            
            fixed4 frag (v2f i) : SV_Target {
                // 准备数据（归一化向量非常重要！）
                float3 worldNormal = normalize(i.worldNormal);
                float3 worldLightDir = normalize(_WorldSpaceLightPos0.xyz); // 指向光源
                float3 viewDir = normalize(_WorldSpaceCameraPos.xyz - i.worldPos); // 指向摄像机
                
                // --- A. 环境光 (Ambient) ---
                fixed3 ambient = UNITY_LIGHTMODEL_AMBIENT.xyz;
                
                // --- B. 漫反射 (Diffuse) ---
                // saturate 把值截取在 [0, 1] 之间，防止背光面变负数
                fixed3 diffuse = _LightColor0.rgb * _Diffuse.rgb * saturate(dot(worldNormal, worldLightDir));
                
                // --- C. 高光 (Specular - Blinn-Phong) ---
                float3 halfDir = normalize(worldLightDir + viewDir); // 半程向量
                fixed3 specular = _LightColor0.rgb * _Specular.rgb * pow(saturate(dot(worldNormal, halfDir)), _Gloss);
                
                // 最终颜色 = 环境光 + 漫反射 + 高光
                return fixed4(ambient + diffuse + specular, 1.0);
            }
            ENDCG
        }
    }
}
```

## 关键点解析

1.  **`Tags { "LightMode"="ForwardBase" }`**：
    这是必须的。告诉 Unity 这个 Pass 处理主光源（通常是平行光）。如果不写，`_LightColor0` 等变量可能无法正确获取。

2.  **`#include "Lighting.cginc"`**：
    包含光照相关的内置变量，如 `_LightColor0`（光源颜色）。

3.  **`normalize()`**：
    **非常重要**。在插值过程中（从顶点到片元），向量长度会发生变化。在片元着色器计算光照前，必须先把方向向量归一化（长度变为1）。

4.  **坐标空间**：
    光照计算通常在 **世界空间 (World Space)** 进行。所以我们在顶点着色器中把法线和位置都转到了世界空间。

## 练习
1.  创建一个球体 (Sphere)，应用这个 Shader。
2.  调节 `Gloss` 值，观察高光范围的变化（值越大，高光越集中，物体越像金属/塑料）。
3.  尝试把 `_Specular` 颜色设为黑色，看看只剩漫反射是什么样。
