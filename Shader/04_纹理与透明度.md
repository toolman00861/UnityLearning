# 04. 纹理与透明度：让物体有图案和半透明

## 1. 纹理采样 (Texture Sampling)

纹理（贴图）是 Shader 中最重要的素材。它决定了物体表面的图案。
要在 Shader 中使用纹理，我们需要：
1.  **定义属性**：`_MainTex ("Texture", 2D) = "white" {}`
2.  **获取 UV 坐标**：模型自带纹理坐标 (`TEXCOORD0`)。
3.  **采样函数**：在片元着色器中使用 `tex2D(_MainTex, uv)` 读取颜色。

## 2. 透明度 (Transparency)

Unity 处理透明物体主要有两种方式：
1.  **Alpha Test (透明度测试/镂空)**：像素要么完全显示，要么完全丢弃（如铁丝网、草）。
2.  **Alpha Blending (透明度混合)**：像素颜色与背景颜色按比例混合（如玻璃、水）。

---

## 实战：带纹理的半透明 Shader

创建一个新 Shader `04_TextureAlpha`，输入以下代码：

```shader
Shader "MyShader/04_TextureAlpha" {
    Properties {
        _MainTex ("Base Texture", 2D) = "white" {} // 纹理属性
        _Color ("Tint Color", Color) = (1, 1, 1, 1) // 叠加颜色
        _AlphaScale ("Alpha Scale", Range(0, 1)) = 1.0 // 透明度系数
    }
    
    SubShader {
        // --- 关键设置 ---
        Tags { 
            "Queue"="Transparent"  // 渲染队列：最后渲染，保证能看到后面的物体
            "RenderType"="Transparent" 
            "IgnoreProjector"="True" 
        }
        
        // 混合模式：SrcAlpha (当前像素透明度) * 当前颜色 + (1 - SrcAlpha) * 背景颜色
        Blend SrcAlpha OneMinusSrcAlpha
        
        // 关闭深度写入：通常半透明物体不写入深度缓冲，避免遮挡后面的物体
        ZWrite Off
        
        Pass {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            
            #include "UnityCG.cginc"
            
            struct appdata {
                float4 vertex : POSITION;
                float2 uv : TEXCOORD0; // 获取 UV 坐标
            };

            struct v2f {
                float4 pos : SV_POSITION;
                float2 uv : TEXCOORD0;
            };

            sampler2D _MainTex; // 对应 Properties 中的 _MainTex
            float4 _MainTex_ST; // 自动生成：用于处理 Tiling 和 Offset (缩放和偏移)
            fixed4 _Color;
            float _AlphaScale;
            
            v2f vert (appdata v) {
                v2f o;
                o.pos = UnityObjectToClipPos(v.vertex);
                
                // TRANSFORM_TEX 是宏，计算 uv * Tiling + Offset
                // 必须配合 _MainTex_ST 变量使用
                o.uv = TRANSFORM_TEX(v.uv, _MainTex);
                
                return o;
            }
            
            fixed4 frag (v2f i) : SV_Target {
                // 1. 采样纹理颜色
                fixed4 texColor = tex2D(_MainTex, i.uv);
                
                // 2. 叠加颜色
                fixed4 finalColor = texColor * _Color;
                
                // 3. 修改透明度 (Alpha)
                finalColor.a = finalColor.a * _AlphaScale;
                
                return finalColor;
            }
            ENDCG
        }
    }
}
```

## 关键点解析

1.  **`sampler2D _MainTex`**：
    对应 ShaderLab 属性中的 `2D` 类型。

2.  **`float4 _MainTex_ST`**：
    这是 Unity 的命名约定。只要声明了纹理 `_Name`，再声明一个 `_Name_ST` (Scale & Translation)，`TRANSFORM_TEX` 宏就能正确处理材质面板上的 Tiling（缩放）和 Offset（偏移）。

3.  **渲染队列 (Render Queue)**：
    *   `"Queue"="Transparent"`：告诉 Unity 把这个物体放在“不透明物体”画完之后再画。如果不加这个，透明物体可能会挡住后面的东西，或者显示顺序错乱。

4.  **混合模式 (Blend)**：
    `Blend SrcAlpha OneMinusSrcAlpha` 是最常用的透明混合公式。
    *   **SrcAlpha**：源颜色（当前计算出的颜色）的透明度。
    *   **OneMinusSrcAlpha**：1 - 源透明度（即背景颜色的权重）。

5.  **ZWrite Off**：
    如果不关闭深度写入，透明物体会像实心物体一样阻挡深度测试，导致它后面的物体（即使透过它应该能看到）被剔除而不显示。

## 练习
1.  找一张带透明通道的 PNG 图片（如树叶、窗户），赋值给 Material 的 Texture。
2.  调节 `Alpha Scale`，观察物体逐渐消失。
3.  尝试修改 Tiling 的 X/Y 值，观察纹理重复的效果。
4.  **进阶**：尝试把 `Blend SrcAlpha OneMinusSrcAlpha` 改为 `Blend One One` (Additive/线性减淡)，看看效果有什么不同（常用于特效，如火焰、光晕，会越叠越亮）。
