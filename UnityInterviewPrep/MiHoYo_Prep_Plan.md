## 第一阶段：笔试与基础夯实 (1-2周)
*目标：确保计算机基础和算法无死角，这是大厂的敲门砖。*

### 1. 算法与数据结构 (重中之重)
米哈游笔试题难度通常在 LeetCode Medium 到 Hard 之间。
- **重点复习**：
  - 动态规划 (DP)：背包问题、路径问题。
  - 图论：最短路径 (Dijkstra/A*)、并查集。
  - 树：二叉树遍历、各种树的操作。
  - 几何数学：向量点乘/叉乘在游戏中的应用、矩阵变换、四元数 (万向节死锁)。
- **对应资料**：`基础/03_算法数据结构深度实战.md` (请确保包含大量手写代码练习)

### 2. C# 与 语言底层
- **重点**：
  - GC 机制 (分代算法、Mark-Sweep)。
  - 装箱拆箱 (IL代码层面)。
  - 闭包的底层实现 (匿名类)。
  - 多线程与异步 (`Task` vs `Thread`, `async/await` 状态机)。
- **对应资料**：`基础/01_CSharp核心深度解析.md` 和 `基础/02_CSharp高级特性与异步.md`

### 3. 操作系统与网络
- **重点**：
  - 进程 vs 线程。
  - 虚拟内存、内存对齐。
  - TCP/UDP 区别 (可靠性、握手挥手)。
  - 帧同步 vs 状态同步 (关键点：预测回滚、插值)。

## 第二阶段：游戏引擎与业务逻辑 (2周)
*目标：展示你能做“复杂系统”和“核心玩法”。*

### 1. 3C 开发 (Character, Camera, Control)
这是JD中明确提到的核心模块，也是动作游戏(ACT/ARPG)的灵魂。
- **准备内容**：
  - **角色状态机**：有限状态机 (FSM) vs 行为树 (Behavior Tree) vs 层次状态机 (HFSM)。
  - **移动处理**：如何处理碰撞、斜坡移动、角色控制器的实现 (CharacterController vs Rigidbody)。
  - **相机系统**：相机跟随、遮挡剔除、弹簧臂 (Spring Arm)、相机震动。

### 2. Unity 引擎深度
- **重点**：
  - **生命周期**：`Awake` -> `OnEnable` -> `Start` -> `FixedUpdate` -> `Update` -> `LateUpdate` 的精确顺序及应用场景。
  - **协程 (Coroutine)**：迭代器原理，如何自己实现一个简单的协程系统。
  - **资源管理**：AssetBundle 依赖打包策略、引用计数管理 (对应你的 `Addressable` 文件夹)。
  - **UI 系统**：UGUI 渲染流程 (Rebuild, Batching)、事件系统原理。

---

## 第三阶段：图形学与性能优化 (米哈游特色) (2周)
*目标：这是米哈游最看重的领域，也是区分度最大的地方。*

### 1. 渲染管线与 Shader
- **重点**：
  - 渲染管线流程 (应用阶段 -> 几何阶段 -> 光栅化阶段)。
  - **PBR (物理渲染)**：了解基本原理 (能量守恒、菲涅尔效应、微表面模型)。
  - **NPR (非真实感渲染)**：卡通渲染 (Toon Shading)、描边算法 (Back face expansion, Post-process depth based)。
  - 阴影技术 (Shadow Map, PCF, CSM)。
- **对应资料**：你的 `Shader/` 文件夹非常好，建议深入 `03_光照模型基础.md` 并补充 NPR 相关知识。

### 2. 性能优化
- **重点**：
  - **CPU 优化**：减少 Draw Call (Static/Dynamic Batching, GPU Instancing)。
  - **内存优化**：减少 GC Alloc，对象池 (Object Pooling) 的正确使用，纹理压缩格式 (ASTC/ETC2)。
  - **GPU 优化**：Overdraw 分析，带宽优化。
  - **工具**：熟练描述 Unity Profiler, Frame Debugger, RenderDoc 的使用经历。
- **对应资料**：`Addressable/Addressables排障与优化手册.md` (资源优化)，需补充渲染和代码优化内容。
