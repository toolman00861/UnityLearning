# 树 (Tree) - 遍历与操作

树结构是游戏场景管理（Quadtree/Octree）、UI 层级、AI 决策树等的基础。

## 1. 核心概念

*   **二叉树 (Binary Tree)**: 每个节点最多有两个子节点。
*   **二叉搜索树 (BST - Binary Search Tree)**: 左子树所有节点值 < 根节点值 < 右子树所有节点值。
*   **平衡二叉树 (AVL Tree / Red-Black Tree)**: 左右子树高度差不超过 1。
*   **满二叉树 (Full Binary Tree)**: 每个节点都有 0 或 2 个子节点。
*   **完全二叉树 (Complete Binary Tree)**: 除了最后一层，其他层都是满的，且最后一层节点靠左。

## 2. 遍历 (Traversal)

### 2.1 深度优先搜索 (DFS)

*   **前序遍历 (Pre-order)**: 根 -> 左 -> 右 (适合序列化、复制树)
    ```csharp
    void PreOrder(TreeNode root) {
        if (root == null) return;
        Visit(root);
        PreOrder(root.left);
        PreOrder(root.right);
    }
    ```
*   **中序遍历 (In-order)**: 左 -> 根 -> 右 (BST 输出有序序列)
    ```csharp
    void InOrder(TreeNode root) {
        if (root == null) return;
        InOrder(root.left);
        Visit(root);
        InOrder(root.right);
    }
    ```
*   **后序遍历 (Post-order)**: 左 -> 右 -> 根 (适合删除树、计算子树信息)
    ```csharp
    void PostOrder(TreeNode root) {
        if (root == null) return;
        PostOrder(root.left);
        PostOrder(root.right);
        Visit(root);
    }
    ```

### 2.2 广度优先搜索 (BFS) / 层序遍历 (Level-order)

使用队列 (Queue) 实现，常用于求树的最小深度、按层输出。

```csharp
void LevelOrder(TreeNode root) {
    if (root == null) return;
    Queue<TreeNode> queue = new Queue<TreeNode>();
    queue.Enqueue(root);

    while (queue.Count > 0) {
        int levelSize = queue.Count;
        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.Dequeue();
            Visit(node);
            if (node.left != null) queue.Enqueue(node.left);
            if (node.right != null) queue.Enqueue(node.right);
        }
    }
}
```

## 3. 常见操作与算法

### 3.1 树的最大深度 (Max Depth)
递归求解：`max(LeftDepth, RightDepth) + 1`

```csharp
int MaxDepth(TreeNode root) {
    if (root == null) return 0;
    return Math.Max(MaxDepth(root.left), MaxDepth(root.right)) + 1;
}
```

### 3.2 验证二叉搜索树 (Validate BST)
不仅要检查左右孩子，还要确保左子树所有节点 < 根 < 右子树所有节点。通常传递 `min` 和 `max` 区间。

```csharp
bool IsValidBST(TreeNode root, long min, long max) {
    if (root == null) return true;
    if (root.val <= min || root.val >= max) return false;
    return IsValidBST(root.left, min, root.val) && IsValidBST(root.right, root.val, max);
}
```

### 3.3 最近公共祖先 (Lowest Common Ancestor - LCA)
**BST**: 利用性质，如果在 `p` 和 `q` 之间，就是 LCA；如果都小于当前节点，往左找；如果都大于，往右找。
**普通二叉树**: 后序遍历。如果左子树找到 `p` 或 `q`，右子树也找到，当前就是 LCA。

```csharp
TreeNode LowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;
    TreeNode left = LowestCommonAncestor(root.left, p, q);
    TreeNode right = LowestCommonAncestor(root.right, p, q);

    if (left != null && right != null) return root; // p, q 分别在左右子树
    return left != null ? left : right; // 都在左子树或右子树
}
```

### 3.4 二叉树的直径 (Diameter)
任意两个节点之间最长路径的长度。
**思路**: 对每个节点，计算 `LeftDepth + RightDepth`，取最大值。注意直径不一定经过根节点。

## 4. 游戏中的应用

1.  **场景图 (Scene Graph)**: 变换矩阵的层级传递 (Parent-Child Transform)。
2.  **空间划分**:
    *   **四叉树 (Quadtree)**: 2D 空间碰撞检测优化。
    *   **八叉树 (Octree)**: 3D 空间划分。
    *   **BVH (Bounding Volume Hierarchy)**: 用于光线追踪加速结构。
3.  **行为树 (Behavior Tree)**: AI 逻辑控制 (Sequence, Selector, Decorator)。
4.  **字典树 (Trie)**: 字符串前缀匹配，如敏感词过滤、自动补全。

## 5. 总结

*   **递归思维**: 树的问题 90% 可以用递归解决。
*   **Base Case**: 永远先处理 `root == null` 的情况。
*   **利用性质**: BST 的中序遍历是有序的，利用这一点可以解决很多问题（如找第 K 小元素）。
