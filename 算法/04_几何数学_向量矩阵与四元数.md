# 几何数学 (Geometric Math) - 向量、矩阵与四元数

游戏开发离不开线性代数，面试中常考察向量运算的几何意义、矩阵变换原理以及旋转表示方法的优缺点。

## 1. 向量 (Vector)

### 1.1 点乘 (Dot Product)
**公式**: `A · B = |A||B|cosθ`
**几何意义**:
*   **投影**: A 在 B 方向上的投影长度。
*   **夹角**: `cosθ = (A · B) / (|A||B|)`。
*   **方向判定**:
    *   `> 0`: 锐角 (前方)
    *   `= 0`: 垂直 (正侧方)
    *   `< 0`: 钝角 (后方)

**应用**:
*   判断敌人在玩家前方还是后方 (`forward · (enemyPos - playerPos) > 0`)。
*   光照计算 (Lambert Diffuse): `N · L` (法线与光线夹角)。

### 1.2 叉乘 (Cross Product)
**公式**: `A × B = |A||B|sinθ * n` (n 为垂直于 A, B 平面的单位向量)
**几何意义**:
*   **法线**: 结果向量垂直于 A 和 B 构成的平面。
*   **左/右判定**: 在 2D 中 (z=0)，通过结果的 z 分量判断相对位置。
    *   `AxB.y > 0`: B 在 A 的顺时针方向 (右侧，视坐标系而定)
    *   `AxB.y < 0`: B 在 A 的逆时针方向 (左侧)

**应用**:
*   计算三角形法线 (Normal Calculation)。
*   判断多边形是顺时针还是逆时针 (Back-face Culling)。
*   构建坐标系 (基向量构建)。

## 2. 矩阵 (Matrix)

### 2.1 变换矩阵 (Transformation Matrix)
通常使用 4x4 矩阵进行仿射变换 (Affine Transformation)。

*   **平移 (Translation)**: 最后一列 (x, y, z)。
*   **旋转 (Rotation)**: 左上角 3x3 矩阵正交。
*   **缩放 (Scale)**: 对角线元素。

**复合变换顺序**: `M = T * R * S` (先缩放，再旋转，最后平移 - 从右向左乘)。
**注意**: 矩阵乘法不满足交换律 (`AB != BA`)。

### 2.2 坐标系变换
*   **MVP 矩阵**: `Model * View * Projection`
    *   **Model**: 本地 -> 世界
    *   **View**: 世界 -> 相机 (观察空间)
    *   **Projection**: 相机 -> 裁剪空间 (Clip Space)

## 3. 四元数 (Quaternion)

### 3.1 为什么使用四元数？
*   **欧拉角 (Euler Angles)**: 直观 (x, y, z 轴旋转)，但存在 **万向节死锁 (Gimbal Lock)** 问题。当两个旋转轴重合时，丢失一个自由度。
*   **矩阵**: 无死锁，但数据量大 (16 floats)，插值困难，且容易累积误差导致非正交。
*   **四元数**: `q = w + xi + yj + zk`。无死锁，插值平滑 (Slerp)，存储紧凑 (4 floats)。

### 3.2 常用操作
*   **单位四元数**: `|q| = 1`，表示纯旋转。
*   **共轭/逆**: `q* = w - xi - yj - zk`。若 q 为单位四元数，`q^-1 = q*`。
*   **旋转向量**: `v' = q * v * q^-1`。
*   **插值 (Slerp - Spherical Linear Interpolation)**: 在球面上平滑插值，保证角速度恒定。
    *   `Lerp`: 线性插值，非匀速，路径是直线（弦）。

### 3.3 万向节死锁 (Gimbal Lock) 详解
*   **现象**: 在欧拉角旋转中，当中间轴旋转 90 度时，第一个轴和第三个轴重合，导致失去一个旋转自由度。
*   **本质**: 欧拉角是按顺序进行的连续旋转，后续旋转轴会受到前序旋转的影响。

## 4. 常见面试题

1.  **如何判断点在三角形内？**
    *   **重心坐标法**: `P = uA + vB + wC`，若 `u, v, w >= 0` 且 `u+v+w=1`。
    *   **叉乘法**: 点 P 必须在三条边的同一侧 (同向法线)。

2.  **AABB vs OBB 包围盒**
    *   **AABB (Axis-Aligned Bounding Box)**: 轴对齐，计算快，但不贴合旋转物体。
    *   **OBB (Oriented Bounding Box)**: 有方向，贴合紧密，但相交检测复杂 (分离轴定理 SAT)。

3.  **齐次坐标 (Homogeneous Coordinates) 的 w 分量有什么用？**
    *   区分点 `(x, y, z, 1)` 和向量 `(x, y, z, 0)`。
    *   透视除法 (Perspective Division): `(x/w, y/w, z/w)` 实现近大远小。

## 5. 总结

*   **向量**: 掌握点乘（夹角/投影）和叉乘（法线/方向）。
*   **矩阵**: 理解 MVP 变换流程，背诵变换顺序。
*   **四元数**: 知道它解决了 Gimbal Lock 问题，并且适合插值。
