# 游戏大厂（米哈游/腾讯/网易）校招应届生画像与要求分析

本文档基于 2024-2025 届校招数据整理，分析了米哈游、腾讯、网易等头部游戏公司对**客户端开发（Unity/Unreal）**岗位的校招要求。

## 1. 核心画像：什么样的应届生能进大厂？

进入头部游戏大厂的应届生通常具备以下 "3+1" 核心竞争力：

### A. 学历与专业（敲门砖）
*   **学历**：通常要求全日制本科及以上，985/211 院校或海外知名高校的学生具有显著优势。
*   **专业**：计算机科学与技术、软件工程、数学、物理等理工科专业为主。
*   **例外**：如果项目经验极强（如由独立开发上架游戏的经历、Github 高 Star 项目），学历门槛会适当放宽。

### B. 硬核技术基础（决定下限）
大厂校招笔试和面试最看重的是**计算机基础**，而非仅仅会用引擎。
1.  **编程语言**：
    *   **C++**：腾讯和网易底层架构多用 C++，这是必考项（虚函数、内存管理、STL、模板等）。
    *   **C#**：Unity 开发岗位的核心，需掌握底层机制（GC、委托/事件、反射、协程）。
2.  **计算机基础**：
    *   **数据结构与算法**：LeetCode Medium/Hard 级别，常见考点包括树、图、堆、动态规划、排序算法。
    *   **操作系统**：进程/线程、死锁、内存管理、虚拟内存。
    *   **计算机网络**：TCP/UDP、HTTP/HTTPS、Socket 编程（特别是帧同步/状态同步相关）。
3.  **图形学（加分项/进阶项）**：
    *   对于米哈游等重度渲染项目，图形学是重要考察点（渲染管线、Shader 编程、线性代数、PBR 理论）。

### C. 实践经验（决定上限）
这是区分“做题家”和“开发者”的关键。
*   **大厂实习**：拥有腾讯、网易、字节、米哈游等公司的实习经历是含金量最高的背书。
*   **完整项目**：不仅是 Demo，而是有完整核心玩法、甚至上架过的游戏。
    *   *优质案例*：复刻经典游戏核心系统、实现一个渲染管线、参与开源游戏引擎贡献。
*   **竞赛奖项**：ACM/ICPC、通过 CCPP、吉比特/网易高校游戏开发大赛获奖者。

### D. 软素质（隐藏关卡）
*   **游戏热情**：不仅仅是爱玩游戏，而是对游戏机制有拆解分析能力（玩过大量 3A 或独立游戏）。
*   **学习能力**：游戏技术迭代快，考察是否关注新技术（DOTS、UE5 Nanite/Lumen、AIGC）。

## 2. 三巨头（米/腾/网）侧重点对比

| 公司 | 关键词 | 侧重倾向 | 面试风格 |
| :--- | :--- | :--- | :--- |
| **腾讯 (Tencent)** | C++、基础扎实 | 极度重视 **C++** 和 **CS 基础**（网络/OS）。即使是 Unity 岗，也可能深挖 C++ 底层。注重逻辑思维和算法潜力。 | 流程标准，笔试难度大，面试轮次多，考察全面。 |
| **米哈游 (miHoYo)** | 图形学、二次元 | 对 **图形学** 要求较高（TA/引擎向）。Unity 技术栈深度要求高。看重对 ACG 文化的理解和热爱（"技术宅拯救世界"）。 | 技术面试硬核，可能会问很深的渲染或优化问题。 |
| **网易 (NetEase)** | 玩法多样、硬核 | 互娱和雷火风格略有不同。笔试题目通常很难（甚至有图形学大题）。看重实际动手能力和对游戏玩法的理解。 | 笔试题量大且难，面试中会考察具体的场景解决能力。 |

## 3. 简历准备与避坑指南

### 优秀简历模板结构
1.  **个人信息**：姓名、联系方式、Github 链接、个人技术博客/作品集链接（**非常重要**）。
2.  **教育背景**：学校、专业、绩点（如果高）、相关课程。
3.  **项目经历**（倒叙，最重要）：
    *   项目名称 | 担任角色 | 技术栈
    *   *STAR 法则描述*：背景(Situation) -> 任务(Task) -> 行动(Action) -> 结果(Result)。
    *   *示例*：独立开发 3D 动作游戏，使用 Unity 引擎。实现状态机管理角色行为（Action），利用对象池优化内存（Result：GC 降低 50%），编写 Shader 实现卡通渲染效果。
4.  **实习经历**：公司、岗位、具体产出。
5.  **技能清单**：C++/C#、Unity、算法、图形学等，标注熟练程度（了解/熟悉/精通）。

### 常见雷区
*   **项目太水**：仅罗列课程作业或网上满大街的教程 Demo（如 "Roll a Ball"）。
*   **技能堆砌**：写了“精通 C++”却答不出虚函数表结构；写了“熟悉图形学”却不懂光栅化流程。
*   **无作品链接**：游戏开发是视觉化的，没有 Demo 视频或可玩包，说服力大打折扣。

## 4. 备战时间表（以 2026 届为例）

*   **大三下/研二下 (3月-5月)**：准备暑期实习。这是转正率最高的黄金机会。
*   **暑期 (7月-8月)**：大厂实习，争取转正 Offer。
*   **秋招提前批 (6月-8月)**：神仙打架，免笔试机会多，主要给头部选手。
*   **秋招正式批 (9月-11月)**：主力战场，海投，笔试面试集中爆发。
*   **春招 (次年3月-5月)**：补录，岗位较少，主要捡漏。

## 5. 推荐学习资源
*   **图形学**：GAMES101 (闫令琪), GAMES202。
*   **C# / Unity**：Unity 官方文档, 《CLR via C#》, Catlike Coding 教程。
*   **C++**：《C++ Primer》, 《Effective C++》。
*   **算法**：LeetCode, 《剑指 Offer》。
